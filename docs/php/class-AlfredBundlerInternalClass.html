Class AlfredBundlerInternalClass

Internal API Class for Alfred Bundler

This class is the only one that you should interact with. The rest of the magic that the bundler performs happens under the hood. Also, the backend of the bundler (here the 'AlfredBundlerInternalClass') may change; however, this wrapper will continue to work with the bundler API for the remainder of this major version.
